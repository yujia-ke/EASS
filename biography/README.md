# biography

用于整理苏轼的生平资料、仕途经历、交游关系与时代背景。

这一部分和 [`timeline/`](../timeline/) 的分工不同：

- `timeline/` 偏时间轴，回答“哪一年发生了什么”
- `biography/` 偏人物传记，回答“这些经历如何连起来，苏轼为什么会成为这样的人”

## 当前结构

- [`overview.md`](./overview.md)：一篇短传，先把苏轼一生讲通
- [`early-years.md`](./early-years.md)：家学背景、少年成长、科举与入仕初期
- [`political-life.md`](./political-life.md)：仕途经历、新法之争与政治命运
- [`huangzhou-huizhou-hainan.md`](./huangzhou-huizhou-hainan.md)：黄州、惠州、海南这条关键人生线
- [`relationships.md`](./relationships.md)：家人、师友、政敌与精神同道
- [`places.md`](./places.md)：重要人生地点及其意义
- [`personality-and-thought.md`](./personality-and-thought.md)：性格、思想与精神世界
- [`turning-points.md`](./turning-points.md)：一生中的关键转折点
- [`sources.md`](./sources.md)：后续可继续补充的传记与资料来源

## 建议阅读顺序

1. 先看 [`overview.md`](./overview.md)，建立整体印象。
2. 再看 [`political-life.md`](./political-life.md) 和 [`huangzhou-huizhou-hainan.md`](./huangzhou-huizhou-hainan.md)，抓住他最关键的人生线。
3. 最后进入 [`relationships.md`](./relationships.md) 与 [`personality-and-thought.md`](./personality-and-thought.md)，从“事情”进入“这个人”。
