# 苏轼书法总览

苏轼不仅是文学史上的核心人物，也是宋代书法新风的重要代表。故宫博物院相关出版资料将苏轼、黄庭坚、米芾、蔡襄并列为“宋四家”，并指出苏轼书法“真率自然”，取法颜真卿而又能自开面目。

对这个仓库来说，整理苏轼书法不只是“看名作”，更重要的是把他的书法放回他的生命经验里去理解。很多最动人的作品，都与他的贬谪、行旅、病后、自遣、写诗作文紧密连在一起。

## 可以先抓住的几个特点

### 1. 书法和人生状态关系很近

苏轼很多重要书迹，并不是为展览或炫技而作，而是在具体处境中自然写出。所以看他的字，往往不能只看结体和笔法，还要看当时他正经历什么。

### 2. 行书最值得先读

如果刚开始整理苏轼书法，优先看行书作品会更容易进入状态。它最能体现苏轼笔下那种自然流动、起伏跌宕、但又不失骨力的感觉。

### 3. 书与文常常不可分

苏轼不少书法作品本身就是诗、赋、尺牍。读内容本身，往往就是理解书法节奏的一部分。

### 4. 不宜只用“豪放”概括

苏轼书法当然有开张、雄健的一面，但也常有沉着、简静、甚至近乎苍冷的气息。尤其在贬谪后的作品里，这种复杂性很明显。

## 建议的学习路径

1. 从 [`representative-works.md`](./representative-works.md) 里选 3 到 5 件作品建立第一印象。
2. 每件作品至少记录一次自己的观察，写在 [`study-template.md`](./study-template.md) 的格式里。
3. 后续再按时期拆分，例如“黄州时期”“岭南时期”“尺牍专题”。
