# 苏轼书法代表作品

这份索引先收录几件较适合入门的作品。标准不是“最全”，而是便于建立苏轼书法的整体印象。

## 1. 《黄州寒食帖》

- 收藏信息：国立故宫博物院相关法帖系统可见
- 书体：行书
- 所处阶段：黄州时期
- 为什么重要：这几乎是理解苏轼书法最绕不过去的一件作品。诗本身写黄州寒食时的处境，字势也随着情绪推进而变化。
- 观察重点：看整篇行气如何从压抑、沉郁逐渐走向更强烈的起伏；也可以留意黄庭坚题跋对它的高度评价。
- 延伸阅读：[`huangzhou-cold-food-post.md`](./huangzhou-cold-food-post.md)

## 2. 《洞庭春色、中山松醪二赋卷》

- 收藏信息：国立故宫博物院公开资料可见相关图版条目，当前仓库使用的是《清三希堂法帖（十一）》中的公开图像
- 书体：行书
- 所处阶段：晚年南迁途中书写
- 为什么重要：这是理解苏轼晚年书风的重要作品之一。内容上借酒赋抒怀，书写时则已在再贬岭南的路上。
- 观察重点：看长卷中的节奏控制、结字紧密感，以及晚年笔墨里兼具老辣和从容的状态。
- 延伸阅读：[`dongting-spring-colors-and-zhongshan-pine-wine.md`](./dongting-spring-colors-and-zhongshan-pine-wine.md)

## 3. 《养生论真迹》

- 收藏信息：国立故宫博物院藏
- 书体：法书册页
- 所处阶段：岭南时期
- 为什么重要：这件作品直接带出苏轼晚年对身心、忧患、养生的关注。文本内容本身就非常值得读。
- 观察重点：先通读文字内容，再看书写是否呈现出更稳定、平和、内收的气质。

## 4. 《次韵三舍人省上诗》

- 收藏信息：见国立故宫博物院《宋四家法书卷》
- 书体：法书卷中的苏轼作品
- 所处阶段：待继续细化
- 为什么重要：它适合作为“苏轼与宋四家并观”的入口。放在同卷中看，更容易比较苏轼与蔡襄、黄庭坚、米芾的差异。
- 观察重点：横向比较同卷他家作品，感受苏轼书风在节奏、体势和气息上的不同。

## 5. 《南轩梦语》《致主簿曹君尺牍》

- 收藏信息：见国立故宫博物院《宋四家书卷》
- 书体：尺牍、短篇文字
- 所处阶段：待继续细化
- 为什么重要：比起长卷名作，这类作品更能看到苏轼日常书写的一面，也更适合做细读。
- 观察重点：看尺牍中的自然感、松紧变化和日常语气，不要只把苏轼书法理解成“大开大合”的名作风格。
