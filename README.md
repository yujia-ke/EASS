# EASS

Everything About Su Shi

EASS 是 `Everything About Su Shi` 的缩写。这个仓库用来系统收集、整理和学习北宋文学家、书法家、政治家苏轼的生平、作品与精神世界，也记录我们在阅读与思考中的个人成长。

## 仓库目标

1. 梳理苏轼的生平轨迹，理解其所处时代与历史处境。
2. 分类整理苏轼的诗、词、文、赋与书法作品，建立可持续扩展的资料库。
3. 沉淀阅读笔记、名句摘录与个人反思，让学习从“知道”走向“内化”。

## 目录结构

- [`biography/`](./biography/)：生平资料、仕途经历、交游关系、时代背景
- [`works/`](./works/)：诗、词、文、赋及相关文本解读
- [`calligraphy/`](./calligraphy/)：书法作品、风格分析、临习笔记
- [`notes/`](./notes/)：读书笔记、观后感、延伸思考
- [`quotes/`](./quotes/)：名句摘录、出处整理、个人理解
- [`timeline/`](./timeline/)：按年份整理的重要经历、作品与历史节点

## 建议的整理方式

### 1. 生平线

优先建立清晰、可信的时间线，把“苏轼在何时、何地、经历了什么”整理清楚。

### 2. 作品线

按体裁与时期双重维度整理作品，例如：

- 诗
- 词
- 散文
- 赋
- 书法题跋

### 3. 自我提升线

将苏轼的思想、气度、处世方式转化成自己的长期笔记，例如：

- 遇到挫折时如何自处
- 如何兼顾理想、现实与责任
- 如何保持审美、幽默与生命力

## 后续可继续补充的内容

1. 为 `timeline/` 年表补全作品对应关系与关键地点。
2. 在 `works/` 中按体裁建立索引文件。
3. 在 `quotes/` 中整理高频名句，并标注出处。
4. 在 `notes/` 中建立你们两人的长期共读笔记。
5. 在 `calligraphy/` 中补充代表作品与学习资料。

## 一句话定位

这是一个围绕苏轼展开的长期学习仓库，也是一个借由苏轼来理解人生、修炼自我的共同项目。
