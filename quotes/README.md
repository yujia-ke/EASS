# quotes

用于摘录苏轼名句，并标注出处、语境与个人理解。

## 建议结构

- [`index.md`](./index.md)：名句总索引，适合快速浏览
- [`template.md`](./template.md)：单条摘录模板，方便复制使用
- 后续可按主题继续拆分，例如：
  - `life.md`
  - `friendship.md`
  - `adversity.md`
  - `freedom.md`

## 摘录原则

1. 尽量标注原文出处，如作品名、体裁、写作时期。
2. 如果句子流传版本不一，保留你们采用的版本，并注明情况。
3. 不只摘录“名句”，也记录为什么打动你们。
4. 优先积累少量高质量摘录，不急着堆数量。

## 推荐字段

- `原句`
- `出处`
- `体裁`
- `大致时期`
- `主题`
- `个人理解`
- `延伸思考`
